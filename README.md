# Kubernetes — Prácticas de Sistemas Distribuidos

Recursos de aprendizaje para desplegar y gestionar aplicaciones en Kubernetes usando **minikube**. Desarrollado para la asignatura de **Sistemas Distribuidos** de la carrera de Ciencias de la Computación.

---


## Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

- [Docker](https://docs.docker.com/get-docker/) — motor de contenedores
- [minikube](https://minikube.sigs.k8s.io/docs/start/) — clúster K8s local
- [kubectl](https://kubernetes.io/docs/tasks/tools/) — CLI de Kubernetes

Verificar instalación:

```bash
docker --version
minikube version
kubectl version --client
```
